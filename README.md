# How to compile
 - In CMD: `pip install pyinstaller` & `pip install pyarmor`
 - Drag the main.py file to the desktop and in CMD: `cd desktop`
 - In CMD: `pyarmor pack -e " --onefile" main.py`, go to dist and grab your packed file!
